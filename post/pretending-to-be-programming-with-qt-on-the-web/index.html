2020-03-17

Many inhouse tools have been created by people who works with C++ or Python alot, for example researchers, pipeline developer etc. These tools are created to perform simple tasks that doesn't require fancy GUI. Not that you can't create fancy GUI with Qt, but frontend people usually prefer to use different frameworks.
